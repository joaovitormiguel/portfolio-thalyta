# Portfólio — Thalyta Cavalli

Site estático (dark mode, foco em diagramação) com galeria de 4 projetos em largura total, empilhados.

## Como abrir
Abra `docs/index.html` direto no navegador (clique duplo) — é estático, não precisa de servidor.
Para servir localmente, opcionalmente: `cd docs && python3 -m http.server 4599` e acesse http://localhost:4599.

## Estrutura
- `docs/index.html` — página única (HTML + CSS + JS embutidos)
- `docs/assets/` — imagens
  - `iprade/`, `hiperzoo/` — screenshots dos sites (geradas dos PDFs em `projetos/`)
  - `cidade-vernacular/`, `bienal/` — imagens baixadas do Behance

## Projetos
